HTML Cheat Sheet

Below is a cheat sheet for some common HTML elements, properties, and functionalities:

Element/Property Description
<!DOCTYPE> Defines the document type and version of HTML
<html> Root element of an HTML page
<head> Contains meta-information about the HTML document
<title> Defines the title of the document
<body> Contains the visible content of the HTML document
<h1> to <h6> Defines headings of different levels
<p> Defines a paragraph
<a> Defines a hyperlink
<img> Embeds an image
<ul> and <li> Defines an unordered list and its items
<ol> and <li> Defines an ordered list and its items
<div> Defines a division or a section in an HTML document
<span> Defines a section in an HTML document
<input> Defines an input control
<form> Defines an HTML form for user input

CSS Cheat Sheet

Below is a cheat sheet for some common CSS properties and functionalities:

Property/Functionality Description
color Specifies the text color
background-color Sets the background color of an element
font-size Sets the size of the font
font-family Sets the font family for text content
font-weight Specifies the weight of the font (e.g., bold)
margin Sets the margin space around an element
padding Sets the padding space around an element
border Adds a border around an element
text-align Aligns text content horizontally
display Specifies the display behavior of an element
position Sets the positioning method of an element
float Specifies whether an element should float to the left, right, or none
z-index Specifies the stack order of an element
opacity Sets the transparency level of an element
transition Adds smooth transition effects to elements

JavaScript Cheat Sheet

Below is a cheat sheet for some common JavaScript functionalities:

Functionality Description
document.getElementById() Returns a reference to the element by its ID
document.querySelector() Returns the first element that matches a specified CSS selector
addEventListener() Attaches an event handler to the specified element
setTimeout() Executes a function after a specified time interval
setInterval() Repeats a function at specified intervals
createElement() Creates a new HTML element
appendChild() Adds a new child node to an element
innerHTML Sets or returns the HTML content of an element
classList Returns the class name(s) of an element
JSON.parse() Converts a JSON string into a JavaScript object
JSON.stringify() Converts a JavaScript object into a JSON string
localStorage Provides an interface to access a browser's local storage
fetch() Initiates a request to a server and returns a Promise
Promise Represents the eventual completion (or failure) of an asynchronous operation and its resulting value
async/await Provides a syntax for writing asynchronous code in a synchronous way
